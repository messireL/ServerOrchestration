# Release Notes — v0.1.43

## Что исправлено
- UI мониторинга теперь корректно читает данные 3x-ui подписки из вложенного `details` в `summary_json`;
- в строке `subscription:` снова выводятся трафик, лимит, срок действия, типы конфигов и другие разобранные поля, если backend их уже сохранил;
- исправлен случай, когда backend данные уже приносил, а фронт показывал почти пустую строку из-за чтения не того уровня JSON.

## Что проверить после обновления
1. Нажать `Проверить 3x-ui`.
2. У AdminVPS в строке `subscription:` должны появиться cfg / used / total / exp вместо голой технической метки.
3. SSL-строка должна остаться рабочей.
