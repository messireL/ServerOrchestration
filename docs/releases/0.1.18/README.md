# Release 0.1.18

## Что сделано
- исправлено отображение ping latency/time в таблицах Dashboard и Проверки;
- усилен парсинг вывода системной утилиты `ping` для разных форматов строки `time`;
- если ping успешен, но latency не распознана из stdout/stderr, используется measured elapsed time выполнения процесса;
- UI теперь показывает latency как `N ms`, а при отсутствии значения — просто `—`.

## Почему это понадобилось
После релиза `0.1.17` ping перестал падать `Internal Server Error`, но latency могла не показываться в таблицах.
