# ServerOrchestration v0.1.34

## Что исправлено
- Переработана таблица «Проверки доступности»: вместо расползающегося текста добавлены компактные плашки статусов и откликов.
- Исправлен hover строки в dark theme: больше нет белой засветки, из-за которой строку было невозможно читать.
- Добавлен fixed layout для status-table и ограничение ширины/переноса для длинных URL и ошибок.
- Заголовки статусов уточнены: `SSH endpoint`, `Web URL`, `Контур` вместо перегруженных и двусмысленных подписей.
- В колонке «Контур» теперь читаются HTTP / 3x-ui console / 3x-ui sub / SSL как отдельные плашки.

## Что проверить
1. Таблица «Проверки доступности» читается в тёмной теме без белых строк.
2. Длинные URL не разъезжают соседние колонки.
3. Ошибки показываются компактно и не ломают строку таблицы.
4. Плашки статусов и откликов видны и читаются на экране без горизонтального хаоса.
