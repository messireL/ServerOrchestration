# Release 0.1.5

## Что добавлено
- более аккуратный и менее скучный web UI
- перенос интерфейса в статические файлы `index.html`, `styles.css`, `app.js`
- web-форма добавления групп
- web-форма добавления серверов
- web-форма привязки сервера к группе
- быстрые сообщения об успехе и ошибках в интерфейсе
- боковые карточки с группами и действиями
- обновлённый dashboard со статусами и alerts

## Зачем это нужно
Этот релиз переводит проект из режима «сервис и API уже есть» в режим «оператор может реально начать заполнять inventory через GUI».
