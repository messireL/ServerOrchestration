# ServerOrchestration v0.1.47 — subscription html profile parsing fix

## Что исправлено
- добит разбор HTML-профиля подписки 3x-ui через табличные строки `<tr><td>label</td><td>value</td></tr>`;
- добавлен более надёжный fallback на текстовый разбор, если HTML пришёл не в чисто табличном виде;
- в сводке 3x-ui subscription теперь приоритетно выводятся данные профиля подписки:
  - traffic used/total
  - remaining
  - expiry / unlimited
  - profile status
- служебные хвосты вроде `html-embedded`/`encoding` теперь не лезут в summary, если уже удалось вытащить полезные данные профиля.

## Что проверить
1. Нажать **Проверить 3x-ui** для сервера с подпиской.
2. Навести курсор на плашку **3x-ui sub**.
3. Убедиться, что в hover/summary есть реальные данные подписки:
   - трафик
   - лимит
   - остаток
   - срок действия
   - статус
4. Убедиться, что вместо технического текста не показывается `html-embedded`/`encoding`.
