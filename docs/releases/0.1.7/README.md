# Release 0.1.7

## Что добавлено
- переработанный UI с левым меню
- разделение dashboard и функциональных разделов
- отдельные экраны: Главная, Серверы, Группы, Проверки, Оповещения, Дальше
- inventory формы встроены в новые разделы
- таблица статусов вынесена в отдельный экран
- alerts вынесены в отдельный экран

## Зачем
Пользователь подтвердил запуск панели и отдельно попросил переработать интерфейс: сделать левое меню и разнести функционал и дашборды по разделам.

## Что изменено по UX
- экран стал менее скучным и менее монолитным
- операционный UI перестал быть одной длинной страницей
- next-step функционал теперь выделен в roadmap-вкладку
