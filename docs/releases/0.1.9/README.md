# Release 0.1.9

## Что изменено
- Добавлено редактирование и удаление серверов через web UI
- Добавлено редактирование и удаление групп через web UI
- Добавлено удаление связей сервер ↔ группа через отдельную таблицу
- Добавлен фильтр/поиск по серверам и группам
- Уточнена роль Ansible в документации проекта
- Актуализированы README, ТЗ, архитектура, регламент релизов и файл переноса

## Цель релиза
Довести inventory до рабочего состояния, чтобы уже можно было массово заводить серверы через GUI перед переходом к следующему блоку проверок и Ansible-действий.
