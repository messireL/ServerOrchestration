# Release 0.1.12

## Что сделано
- Полностью переработан layout панели.
- Уменьшен и успокоен branding-блок в sidebar.
- Убраны дублирующиеся кнопки и визуальный шум.
- Главная страница собрана как компактная operational-сводка.
- Разделы Серверы, Группы, Проверки и Оповещения приведены к единому стилю.
- Переписан frontend-код страницы, чтобы исключить падения на `null`-элементах при загрузке.

## Зачем
Предыдущий интерфейс визуально и структурно мешал работе. В этом релизе приоритет — аккуратный shell, читаемая типографика и предсказуемый UI-каркас перед следующим шагом по функционалу.
