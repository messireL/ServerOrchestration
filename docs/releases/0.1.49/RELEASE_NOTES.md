# ServerOrchestration v0.1.49 — static cache bust + visible frontend errors

## Что исправлено
- добавлен cache-bust для `styles.css` и `app.js` через query version `?v=0.1.49`;
- для `/` и `/static/*` добавлены `Cache-Control: no-store`, чтобы браузер не держал старый фронтенд после релиза;
- добавлен видимый перехват frontend-ошибок и unhandled promise rejection, чтобы страница больше не умирала молча без реакции кнопок.

## Что проверить
1. После обновления открыть панель обычным обновлением страницы.
2. Проверить, что кнопки снова нажимаются.
3. Если фронтенд всё же падает, на экране должна появиться явная ошибка, а не тишина.
