# Release 0.1.22

- UI-полировка inventory и мониторинга без изменения backend API;
- sidebar разбит на более понятные рабочие секции;
- добавлена оперативная сводка в sidebar;
- на экране серверов добавлены быстрые фильтры;
- статусы ping/SSH/HTTP отображаются отдельными бейджами;
- в списке серверов видны `last_check` и краткий `last_error`;
- на экране проверок добавлена summary-панель по результатам probe;
- roadmap в UI приведён к нормальной последовательности версий.
