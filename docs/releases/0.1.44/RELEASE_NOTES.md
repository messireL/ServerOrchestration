# Release Notes — v0.1.44

## Что исправлено
- добавлен разбор HTML-страницы профиля подписки 3x-ui, которую браузер показывает по subscription URL;
- из HTML теперь вытягиваются: ID подписки, статус, загружено, отправлено, использование, общий лимит, остаток, время последнего онлайна и срок действия;
- трафик и лимит переводятся в байты и попадают в мониторинг как нормальные числовые поля, а не как декоративная боль;
- UI строки `3x-ui sub` теперь показывает не только usage/limit, но и остаток, а для бессрочной подписки пишет `exp ∞`.

## Что проверить после обновления
1. Нажать `Проверить 3x-ui`.
2. Для AdminVPS в строке `3x-ui sub` должны появиться usage/limit/rem/exp.
3. Если подписка бессрочная, должно быть `exp ∞`.
4. Ошибка `subscription endpoint returned HTML instead of subscription data` больше не должна появляться для этой страницы профиля.
