# Release Notes — v0.1.45

## Что исправлено
- исправлен приоритет разбора HTML подписки 3x-ui: если страница одновременно содержит embedded-ссылки и HTML-профиль подписки, теперь профиль **не теряется**;
- `html-embedded` теперь может идти вместе с дополнительными полями профиля: traffic / limit / remaining / status / expiry / last online / subscription id;
- устранён случай, когда парсер слишком рано возвращал результат по найденным URI и не доходил до таблицы профиля.

## Что проверить после обновления
1. Нажать `Проверить 3x-ui`.
2. В ответе `xui_subscription.details` должны остаться `encoding: html-embedded`, но одновременно появиться поля профиля подписки.
3. В UI у `3x-ui sub` должны появиться usage / limit / rem / exp.
