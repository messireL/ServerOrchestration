# Release Notes — v0.1.41

## Что исправлено
- исправлен ложный сценарий `3x-ui monitoring disabled`: backend теперь использует корректный флаг `has_3xui` и подстраховывается наличием console/subscription URL;
- status-плашки на экране мониторинга укорочены, а подробности вынесены в hover/title, чтобы текст не разъезжался по строке;
- SSL- и 3x-ui-плашки теперь показывают подробности по наведению, без простыни внутри таблицы.

## Что проверить после обновления
1. Нажать `Проверить 3x-ui` и убедиться, что пропало `3x-ui monitoring disabled`.
2. Навести мышку на badge `SSL` и проверить, что во всплывающей подсказке есть CN / self-signed / срок.
3. Навести мышку на badge `3x-ui sub` и проверить, что во всплывающей подсказке есть HTTP/status, трафик и expiry, если сервер это отдаёт.
