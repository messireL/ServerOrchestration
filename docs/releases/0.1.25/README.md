# Release v0.1.25

## Что вошло
- доведён полный 3x-ui контур в релиз, без ручных правок:
  - поля сервера `console_3xui_url` и `subscription_3xui_url`;
  - отдельные 3x-ui статусы, HTTP-коды и latency в `server_status`;
  - отдельные настройки `xui_interval_seconds` и `xui_timeout_seconds`;
  - отдельный scheduler-run для 3x-ui;
- backend сам выполняет schema-sync старой БД до нового состояния;
- обновлены release-docs и файл переноса в новый чат.

## Главный смысл релиза
Убрать разрыв между UI/API и БД, чтобы 3x-ui функционал был действительно рабочим после обычной выкладки через Portainer, без ручного SQL и без добивания поломанных колонок на сервере.
