# Deploy / Update v0.1.25

## Основной сценарий
- обновить Git-репозиторий проекта;
- в Portainer выполнить update существующего stack;
- дождаться пересоздания контейнеров;
- проверить backend logs, `/health`, `/version`.

## Что важно в этом релизе
- ручные SQL-правки не нужны;
- новая схема БД доводится backend-ом автоматически при старте;
- если после обновления UI открылся, но поля 3x-ui не сохраняются, значит выкатилась не эта версия кода.
