# Bugs 0.1.17

## Исправлено
- ping probe мог завершаться `Internal Server Error`, если рабочая БД была создана старым релизом без колонки `server_status.summary_json`;
- версия релиза в API/UI могла расходиться с `VERSION`, потому что backend default `APP_VERSION` оставался на `0.1.15`.

## Осталось на следующие релизы
- переработать интерфейс и разнести рабочие экраны по отдельным разделам;
- добавить scheduler проверок и историю probe-run;
- затем перейти к 3x-ui и SSL-проверкам.
