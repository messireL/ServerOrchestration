Архитектурные решения

Коротко и по делу — чтобы новый чат не начинался с археологии и спиритизма.

  • GitHub — основной источник кода проекта.
  • Portainer — основной deploy/update.
  • Docker Compose — обязательный формат поставки.
  • Приложение внутри контейнеров работает non-root.
  • Ping и быстрые probes идут отдельным слоем, не через Ansible.
  • Ansible остаётся исполнительным контуром для операций изменения.
