# ServerOrchestration

Web-панель мониторинга и операционного управления Ubuntu/Linux-серверами.

Текущий подготовленный релиз: **v0.1.52**.
Последний подтверждённый пользователем рабочий релиз: **v0.1.38**.

## Что уже есть в v0.1.52
- всё из v0.1.51 по живому backend-парсингу HTML/JS подписок 3x-ui;
- строка подписки в UI теперь показывается на русском и в нормальном виде;
- если профиль подписки уже распарсен, UI больше не тащит в строку мусор вроде `HTTP 200`, `html embedded`, `cfg`, куски `profile_title` и прочую техдиагностику;
- в карточках/таблицах остаются только полезные поля: трафик, остаток, срок, статус, last seen, subscription id;
- обновлена сопутствующая документация релиза.

## Архитектурный стек
- FastAPI backend
- PostgreSQL
- статический frontend внутри backend
- Docker / Docker Compose
- Portainer как основной путь deploy/update
- Ansible как будущий исполнительный слой для операций изменения состояния сервера
